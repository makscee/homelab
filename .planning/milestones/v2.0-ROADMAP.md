# v2.0 — Claude Code Usage Monitor (CLOSED WITH PIVOT)

**Closed:** 2026-04-16
**Disposition:** Phases 05-07 operational, 08-11 pivoted into v3.0.
**Successor:** v3.0 Unified Stack Migration — custom Next.js admin dashboard absorbing VoidNet + Animaya + homelab into one TypeScript stack.

## Why the pivot

Mid-Phase-05 realization:
1. Feasibility already proven by production exporter running ad-hoc outside GSD.
2. A custom Next.js dashboard with admin features for VoidNet + Animaya + homelab in one UI is more valuable than a Grafana-only solution.
3. Unifying all 3 projects on one TypeScript stack unblocks much faster AI-agent development.

## Tech debt carried to v3.0

From Phase 06/07 operational short-circuits:
- Exporter ran as root (should be uid 65534) — **resolved in v3.0 Phase 13-02**
- Exporter bound 0.0.0.0:9101 (should be 100.101.0.9:9101) — **resolved in v3.0 Phase 13-02**
- `promtool check metrics` never run against production `/metrics`
- Prometheus scrape config not captured in repo
- Exponential backoff on 429 never load-tested (0% 429 in prod so far)
