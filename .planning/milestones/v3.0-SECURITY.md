## Deferred to v3.1 (explicit)

- **SEC-01 / D-22-06** — Caddy per-IP rate-limit on `/api/auth/*`. `caddyserver.com/api/download` upstream build API returned 0-byte responses from both mcow and nether on 2026-04-21. Operator chose defer over `xcaddy` self-build. Pickup path: install `xcaddy`, rebuild Caddy with `caddyserver/rate-limit`, add handler stanza to `servers/mcow/Caddyfile` — tracked as SEC-01 in v3.1.
- **SEC-11** — Strict CSP (drop `unsafe-inline` from `script-src`/`style-src`). Current CSP keeps `unsafe-inline` for React 19 + Next.js 15 compatibility; nonce-based strict CSP requires audit of all inline styles. `header-audit.sh` downgrades to WARN not FAIL for this entry. Tracked as SEC-11 in v3.1.
