  <action>
    1. Collect live state into an evidence file committed to the repo:
       ```bash
       ssh root@mcow "
         echo '## is-active'; systemctl is-active voidnet-bot voidnet-api voidnet-overseer voidnet-satellite 2>&1 || true
         echo '## is-enabled'; systemctl is-enabled voidnet-bot voidnet-api voidnet-overseer voidnet-satellite 2>&1 || true
         echo '## list-unit-files'; systemctl list-unit-files 'voidnet-*.service' --no-pager
         echo '## ps'; ps aux | grep -E 'voidnet' | grep -v grep || true
         for unit in voidnet-bot voidnet-api voidnet-overseer voidnet-satellite; do
           echo \"## cat-${unit}\"
           cat /etc/systemd/system/${unit}.service 2>/dev/null || cat /lib/systemd/system/${unit}.service 2>/dev/null || echo 'NOT FOUND'
         done
       " > servers/mcow/systemd-audit.txt
       ```
    2. For each unit file whose live content (from the `cat-${unit}` section) differs from the repo copy:
       - If the live file exists: REPLACE the repo `.service` file with the live content.
       - If the live file is `NOT FOUND`: keep the repo copy but add an `# STALE (not installed on mcow as of $(date +%F))` comment at top AND note it for flagging in Task 2.
    3. Also check whether a VoidNet portal service exists (Open Question #3 from RESEARCH.md):
       ```bash
       ssh root@mcow "systemctl list-units --type=service --state=running --no-pager | grep -i void || true; ls /opt/voidnet/ 2>/dev/null || true"
       ```
       Append output to `servers/mcow/systemd-audit.txt` under `## portal-probe`. If a `voidnet-portal.service` (or similar) is found, create `servers/mcow/voidnet-portal.service` from the live file content in the same style as bot/api.
    4. Capture the `/opt/voidnet/` layout for README:
       ```bash
       ssh root@mcow "ls -la /opt/voidnet/ 2>/dev/null; file /opt/voidnet/* 2>/dev/null" \
         >> servers/mcow/systemd-audit.txt
       ```
  </action>
  <verify>
    <automated>test -s servers/mcow/systemd-audit.txt && grep -q '## is-active' servers/mcow/systemd-audit.txt && grep -q '## is-enabled' servers/mcow/systemd-audit.txt && grep -q '## portal-probe' servers/mcow/systemd-audit.txt</automated>
  </verify>
