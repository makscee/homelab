## Motivation (why migrate now)

| Option | Description | Selected |
|--------|-------------|----------|
| Resource isolation | Jellyfin transcode/scan chews CPU/RAM on busy docker-tower | ✓ |
| Independent lifecycle | Restart/update Jellyfin without bouncing whole stack | |
| Cleaner iGPU/device ownership | Give /dev/dri to a single LXC | ✓ |
| Other reason | Free text | ✓ |

**User's choice:** Cleaner iGPU + resource isolation + *"for a chance that it will stop buffering so much"*.
**Notes:** Buffering relief is the user-visible success signal.
