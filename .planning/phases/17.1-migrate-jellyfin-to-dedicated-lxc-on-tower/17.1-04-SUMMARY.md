# Phase 17.1 Plan 04: Tower Ingress Swap to CT 101 — Summary

Flipped external Jellyfin ingress from docker-tower to CT 101 via iptables DNAT on `vmbr0:8096`, then unblocked reply-path routing by disabling Tailscale `accept-routes` on CT 101 (prevented `10.10.20.0/24 dev tailscale0` from shadowing the L2-local route). External users back online.

## What Shipped

- **DNAT flip (earlier in 17.1-04, commits `e652c42`, `cbc98b4`):** `vmbr0:8096 → 10.10.20.11:8096`, persisted in `/etc/iptables/rules.v4`.
- **Reply-path fix (this resumption):** `pct exec 101 -- tailscale set --accept-routes=false` (stateful, no re-auth; survives `systemctl restart tailscaled` and reboot via `/var/lib/tailscale/tailscaled.state`).
- **IaC:** `servers/tower/ingress/jellyfin.iptables` (canonical DNAT rule) + `servers/tower/ingress/README.md` (documents the ingress handler shape and the leaf-LXC Tailscale pitfall for future service containers on vmbr1).
- **Incident record:** `17.1-04-INGRESS-LOG.md` — ROLLBACK block intact at top; new "Verification: PASSED" section appended with probe outputs and persistence evidence.

## Deviations from Plan

### Rule 3 — Auto-fixed blocking issue: reply-path routing on CT 101

- **Found during:** Task 3 end-to-end verification (original Plan 04 run)
- **Issue:** Tailscale `accept-routes=true` absorbed a tailnet-wide `10.10.20.0/24` advertisement into CT 101's table 52, sending Jellyfin reply packets out `tailscale0` instead of `eth0`. External users saw TCP timeouts.
- **Fix:** `tailscale set --accept-routes=false` (Option A — operator-approved path via Rule 4 escalation in the INGRESS-LOG; this plan resumption executed the approved fix).
- **Files modified:** CT 101 tailscale prefs (runtime state, persisted in `/var/lib/tailscale/tailscaled.state`).
- **Commit:** this plan's metadata commit.

### Plan-shape deviation (documented at top of INGRESS-LOG)

Original Plan 04 assumed an L7 proxy on `:22098`. Probing proved tower has no nginx/caddy/haproxy; ingress is pure L4 DNAT on `vmbr0:8096`. Artifacts adapted: `jellyfin.conf` (nginx) → `jellyfin.iptables` (rule snippet) + `README.md`.
